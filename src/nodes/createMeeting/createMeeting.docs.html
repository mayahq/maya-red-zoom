Description

Maya node to create or schedule a Zoom Meeting on the user account configured with Maya Virtual Assistant app on Zoom Marketplace.

Config

userId str
Zoom user Id for the user account on which the meeting is to be created. It is set to 'me' as default for self account.
meetingType select str
Zoom Meetings allow four types of meeting to be created. This configuration allow selection between one of the following
  • Instant
  • Scheduled
  • Recurring with no fixed time
  • Recurring with fixed time
Default selection is "Instant"
topic str
Configuration which sets the title or header for the meeting as read by user and participants
startTime str
Time in ISO string format string at which is to be started
timeZone str
Timezone in which the meeting is scheduled. Is a required property and to be handled carefully for multi timezone participants of the meeting. Takes the standard string zone/city format. Defaults to local timezone of Maya device on which the node is running
duration num
Configuration to set the duration of meeting in minutes

Outputs

payload object
Object containing the Zoom URL for meeting invite
