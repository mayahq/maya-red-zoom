Description

List Meetings node is used to fetch all scheduled meeting your account is hosting or invited to on Zoom Meetings.

Config

userId string
The account user id for the account from which the scheduled meeting list is to be fetched
meetingType string
Mandatory selection option to pick from the type of meetings from upcoming, scheduled or live. Default selection is "upcoming"
pageSize num
Configuration to choose number of meetings to be fetched per call. Defaults to 10.

Outputs

payloadobject
The JSON object response containing meeting
